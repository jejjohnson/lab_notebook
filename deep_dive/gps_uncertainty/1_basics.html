Gaussian Processes

Definition

The first thing to understand about GPs is that we are actively placing a distribution P(f)\mathcal{P}(f)P(f) on functions fff where these functions can be infinitely long function values f=[f1,f2,…]f=[f_1, f_2, \ldots]f=[f1​,f2​,…]. A GP generalizes the multivariate Gaussian distribution to infinitely many variables.

A GP is a collection of random variables f1,f2,…f_1, f_2, \ldotsf1​,f2​,…, any finite number of which is Gaussian distributed.

A GP defines a distribution over functions p(f)p(f)p(f) which can be used for Bayesian regression. (Zhoubin)

Another nice definition is:

Gaussian Process: Any set of function variables {fn}n=1N\{f_n \}^{N}_{n=1}{fn​}n=1N​ has a joint Gaussian distribution with mean function mmm. (Deisenroth)

The nice thing is that this is provided by a mean function μ\muμ and covariance matrix K\mathbf{K}K

Bayesian Inference Problem

Objective

Let's have some data set, D={(xi,yi)i=1N}=(X,y)\mathcal{D}= \left\{ (x_i, y_i)^N_{i=1} \right\}=(X,y)D={(xi​,yi​)i=1N​}=(X,y)

Model

yi=f(xi)+ϵif∼GP(⋅∣0,K)ϵi∼N(⋅∣0,σ2)\begin{aligned} y_i &= f(x_i) + \epsilon_i \\ f &\sim \mathcal{GP}(\cdot | 0, K) \\ \epsilon_i &\sim \mathcal{N}(\cdot | 0, \sigma^2) \end{aligned} yi​fϵi​​=f(xi​)+ϵi​∼GP(⋅∣0,K)∼N(⋅∣0,σ2)​

The prior on fff is a GP distribution, the likelihood is Gaussian, therefore the posterior on fff is also a GP (P(f∣D)∝P(D∣f)P(f)=GP∝G⋅GPP(f|\mathcal{D}) \propto P(\mathcal{D}|f)P(f) = \mathcal{GP \propto G \cdot GP}P(f∣D)∝P(D∣f)P(f)=GP∝G⋅GP).

So we can make predictions:

P(y∗∣x∗,D)=∫P(y∗∣x∗,D)P(f∣D)dfP(y_*|x_*, \mathcal{D}) = \int P(y_*|x_*, \mathcal{D})P(f|\mathcal{D})df P(y∗​∣x∗​,D)=∫P(y∗​∣x∗​,D)P(f∣D)df

We can also do model comparison by way of the marginal likelihood (evidence) so that we can compare and tune the covariance functions

P(y∣X)=∫P(y∣f,X)P(f)dfP(y|X) = \int P(y|f,X)P(f)df P(y∣X)=∫P(y∣f,X)P(f)df

Bayesian Treatment

So now how does this look in terms of the Bayes theorem:

P(f∣X,y)=P(y∣f,X)P(f∣X,θ)P(y∣X)\mathcal{P}(f|X,y) = \frac{\mathcal{P}(y|f, X) \mathcal{P}(f|X, \theta)}{\mathcal{P}(y| X)} P(f∣X,y)=P(y∣X)P(y∣f,X)P(f∣X,θ)​

where:

  • Prior:
    • P(f∣X,θ)=GP(μ,Kθ)\mathcal{P}(f|X, \theta)=\mathcal{GP}(\mu, \mathbf{K}_\theta)P(f∣X,θ)=GP(μ,Kθ​)
  • Likelihood (noise model):
    • P(y∣f,X)=N(y∣f(x),σn2I)\mathcal{P}(y|f,X)=\mathcal{N}(y|f(x), \sigma_n^2\mathbf{I})P(y∣f,X)=N(y∣f(x),σn2​I)
  • Marginal Likelihood (Evidence):
    • P(y∣X)=∫fP(y∣f,X)P(f∣X)df\mathcal{P}(y|X)=\int_f \mathcal{P}(y|f,X)\mathcal{P}(f|X)dfP(y∣X)=∫f​P(y∣f,X)P(f∣X)df
  • Posterior:
    • P(f∣X,y)=GP(μ∗,K∗)\mathcal{P}(f|X,y) = \mathcal{GP}(\mu_*, \mathbf{K}_*)P(f∣X,y)=GP(μ∗​,K∗​)

GP Prior

This is the basis of the GP method. Under the assumption that we mentioned above:

p(f∣X,θ)=GP(μ,Kθ)p(f|X, \theta)=\mathcal{GP}(\mu, \mathbf{K}_\theta) p(f∣X,θ)=GP(μ,Kθ​)

Shorthand Notation:

  • P(f)=GP(μ,Kθ)\mathcal{P}(f)=\mathcal{GP}(\mu, \mathbf{K}_\theta)P(f)=GP(μ,Kθ​)

where:

  • μ\muμ is a mean function
  • K\mathbf{K}K is a covariance function

We kind of treat these functions as a vector of function values up to infinity in theory f=[f1,f2,…]f=[f_1, f_2, \ldots]f=[f1​,f2​,…]. But in particular we look at the distribution over the function values, for example fi=f(xi)f_i=f(x_i)fi​=f(xi​). So let's look at the joint distribution between NNN function values fNf_NfN​ and all other function values f∞f_\inftyf∞​. This is 'normally distributed' so we can write the joint distribution roughly as:

P(fN,f∞)=N([μNμ∞],[ΣNNΣN∞ΣN∞⊤Σ∞∞])\mathcal{P}(f_N, f_\infty)=\mathcal{N} \left(\begin{bmatrix} \mu_N \\ \mu_\infty \end{bmatrix}, \begin{bmatrix} \Sigma_{NN} & \Sigma_{N\infty} \\ \Sigma_{N\infty}^{\top} & \Sigma_{\infty\infty} \end{bmatrix}\right)P(fN​,f∞​)=N([μN​μ∞​​],[ΣNN​ΣN∞⊤​​ΣN∞​Σ∞∞​​])

where ΣNN∈RN×N\Sigma_{NN}\in \mathbb{R}^{N\times N}ΣNN​∈RN×N and Σ∞∞∈R∞×∞\Sigma_{\infty\infty} \in \mathbb{R}^{\infty \times \infty}Σ∞∞​∈R∞×∞ (or m→∞m\rightarrow \inftym→∞) to be more precise.

So again, any marginal distribution of a joint Gaussian distribution is still a Gaussian distribution. So if we integrate over all of the functions from the infinite portion, we get:

P(fN)=∫f∞P(fN,f∞)df∞=N(μfN,ΣNN)\begin{aligned} \mathcal{P}(f_N) &= \int_{f_\infty}\mathcal{P}(f_N,f_\infty)df_\infty \\ &= \mathcal{N}(\mu_{f_N},\Sigma_{NN}) \end{aligned}P(fN​)​=∫f∞​​P(fN​,f∞​)df∞​=N(μfN​​,ΣNN​)​

We can even get more specific and split the fNf_NfN​ into training ftrainf_{\text{train}}ftrain​ and testing ftestf_{\text{test}}ftest​. It's simply a matter of manipulating joint Gaussian distributions. So again, calculating the marginals:

P(ftrain,ftest)=∫f∞P(ftrain,ftest,f∞)df∞=N([ftrainftest],[Σtrain×trainΣtrain×testΣtrain×test⊤Σtest×test])\begin{aligned} \mathcal{P}(f_{\text{train}}, f_{\text{test}}) &= \int_{f_\infty}\mathcal{P}(f_{\text{train}}, f_{\text{test}},f_\infty)df_\infty \\ &= \mathcal{N} \left(\begin{bmatrix} f_{\text{train}} \\ f_{\text{test}} \end{bmatrix}, \begin{bmatrix} \Sigma_{\text{train} \times \text{train}} & \Sigma_{\text{train} \times \text{test}} \\ \Sigma_{\text{train} \times \text{test}}^{\top} & \Sigma_{\text{test} \times \text{test}} \end{bmatrix}\right) \end{aligned}P(ftrain​,ftest​)​=∫f∞​​P(ftrain​,ftest​,f∞​)df∞​=N([ftrain​ftest​​],[Σtrain×train​Σtrain×test⊤​​Σtrain×test​Σtest×test​​])​

and we arrive at a joint Gaussian distribution of the training and testing which is still normally distributed due to the marginalization.

Now, something a bit more practical, generally speaking when we program the sampling portion of the prior, we need data. The kernel function is as is and has already been defined with its appropriate parameters. Furthermore, we already have defined the mean function μ\muμ when we initialized the mean function above. So we just need to pass the function through the multivariate normal function along with the number of samples we would like to draw from the prior.

def sample_prior(self, X, n_samples=1, random_state=None):
    """Draws random n_samples from the multivariate normal
    distribution:
    X ~ N(mu, K(X,X))
    
    Parameters
    ----------
    X : array, (N x D)
        The vector data use to draw the random samples from
        where N is the number of data and D is the dimension.
    
    n_samples : int (default = 1)
        The number of random samples to draw from the normal
        distribution
    
    random_state : int, (default = None)
        Needed if we want to specify the random seed used to 
        draw the random samples.

    Returns
    -------
    samples : array, (n_samples x N)
        The samples drawn from the multivariate normal
        distribution.
    """
    # calculate the covariance
    cov = self.kernel(X)

    # handle the random state
    rng = check_random_state(random_state)

    # draw samples from random multivariate normal
    samples = rng.multivariate_normal(self.mu.ravel(), cov, int(n_samples))

    return samples

P(y∣f,X)=∏i=1NN(yi∣fi,σϵ2)=N(y∣f(x),σϵ2IN)\mathcal{P}(y|f,X)=\prod_{i=1}^{N}\mathcal{N}(y_i|f_i,\sigma_\epsilon^2)= \mathcal{N}(y|f(x), \sigma_\epsilon^2\mathbf{I}_N) P(y∣f,X)=i=1∏N​N(yi​∣fi​,σϵ2​)=N(y∣f(x),σϵ2​IN​)

This comes from our assumption as stated above from y=f(x)+ϵy=f(x)+\epsilony=f(x)+ϵ.

Marginal Likelihood (Evidence)

P(y∣X,θ)=∫fP(y∣f,X)⋅P(f∣X,θ)df\mathcal{P}(y|X, \theta)=\int_f \mathcal{P}(y|f,X)\cdot\mathcal{P}(f|X, \theta)df P(y∣X,θ)=∫f​P(y∣f,X)⋅P(f∣X,θ)df

where:

  • P(y∣f,X)=N(y∣f,σn2I)\mathcal{P}(y|f,X)=\mathcal{N}(y|f, \sigma_n^2\mathbf{I})P(y∣f,X)=N(y∣f,σn2​I)
  • P(f∣X,θ)=N(f∣μ,Kθ)\mathcal{P}(f|X, \theta)=\mathcal{N}(f|\mu, K_\theta)P(f∣X,θ)=N(f∣μ,Kθ​)

Note that all we're doing is simply describing each of these elements specifically because all of these quantities are Gaussian distributed.

P(y∣X,θ)=∫fN(y∣f,σn2I)⋅N(f∣μ,Kθ)df\mathcal{P}(y|X, \theta)=\int_f \mathcal{N}(y|f, \sigma_n^2\mathbf{I})\cdot \mathcal{N}(f|\mu, K_\theta)df P(y∣X,θ)=∫f​N(y∣f,σn2​I)⋅N(f∣μ,Kθ​)df

So the product of two Gaussians is simply a Gaussian. That along with the notion that the integral of all the functions is a normal distribution with mean μ\muμ and covariance KKK.

P(y∣X,θ)=N(y∣μ,Kθ+σn2I)\mathcal{P}(y|X, \theta)=\mathcal{N}(y|\mu, K_\theta + \sigma_n^2 \mathbf{I}) P(y∣X,θ)=N(y∣μ,Kθ​+σn2​I)

Posterior

Alternative Notation:

  • P(f∣y)∝N(y∣f,σn2I)⋅N(f∣μ,Kff)\mathcal{P}(f|y)\propto \mathcal{N}(y|f, \sigma_n^2\mathbf{I})\cdot \mathcal{N}(f|\mu, \mathbf{K}_{ff})P(f∣y)∝N(y∣f,σn2​I)⋅N(f∣μ,Kff​)
def posterior(self, X):

    # K(x,x)
    K = self.kernel(self.x_train, self.x_train)

    # K(x,x')
    K_x = self.kernel(self.x_train, X)

    # K(x',x')
    K_xx = self.kernel(X, X)

    # Inverse of kernel
    K_inv = np.linalg.inv(
        K + self.noise_variance * np.eye(len(self.x_train))
    )

    # Calculate the weights
    alpha = K_inv @ self.y_train

    # Calculate the mean function
    mu = K_x @ alpha

    # Calculate the covariance function
    cov = K_xx - K_x.T @ K_inv @ K_x

    return mu, cov

Joint Probability Distribution

To make GPs useful, we want to actually make predictions. This stems from the using the joint distribution of the training data and test data with the formula shown above used to condition on multivariate Gaussians. In terms of the GP function space, we have

P([ff∗])=N([μμ∗],[KxxKx∗K∗xK∗∗])\begin{aligned} \mathcal{P}\left(\begin{bmatrix}f \\ f_*\end{bmatrix} \right) &= \mathcal{N}\left( \begin{bmatrix} \mu \\ \mu_* \end{bmatrix}, \begin{bmatrix} K_{xx} & K_{x*} \\ K_{*x} & K_{**} \end{bmatrix} \right) \end{aligned}P([ff∗​​])​=N([μμ∗​​],[Kxx​K∗x​​Kx∗​K∗∗​​])​

Then solving for the marginals, we can come up with the predictive test points.

P(f∗∣X∗,y,X,θ)=N(f∗∣μ∗,ν∗2)\mathcal{P}(f_* |X_*, y, X, \theta)= \mathcal{N}(f_* | \mu_*, \nu^2_* ) P(f∗​∣X∗​,y,X,θ)=N(f∗​∣μ∗​,ν∗2​)

where:

  • μ∗=K∗(K+σ2I)−1y=K∗α\mu*=K_* (K + \sigma^2 I)^{-1}y=K_* \alphaμ∗=K∗​(K+σ2I)−1y=K∗​α
  • ν∗2=K∗∗−K∗(K+σ2I)−1K∗⊤\nu^2_*= K_{**} - K_*(K + \sigma^2I)^{-1}K_*^{\top}ν∗2​=K∗∗​−K∗​(K+σ2I)−1K∗⊤​

Learning in GPs

The prior m(x),Km(x), Km(x),K have hyper-parameters θ\thetaθ. So learning a GP\mathcal{GP}GP implies inferring hyper-parameters from the model.

P(Y∣X,θ)=∫P(Y∣f)P(f∣X,θ)df\mathcal{P}(Y|X,\theta)=\int \mathcal{P}(Y|f)\mathcal{P}(f|X, \theta)df P(Y∣X,θ)=∫P(Y∣f)P(f∣X,θ)df

However, we are not interested in fff directly. We can marginalize it out via the integral equation. The marginal of a Gaussian is Gaussian.

Note: Typically we use the log⁡\loglog likelihood instead of a pure likelihood. This is purely for computational purposes. The log⁡\loglog function is monotonic so it doesn't alter the location of the extreme points of the function. Furthermore we typically minimize the −log⁡-\log−log instead of the maximum log⁡\loglog for purely practical reasons.

One way to train these functions is to use Maximum A Posterior (MAP) of the hyper-parameters

θ∗=argmaxθlog⁡P(y∣X,θ)=argmaxθlog⁡N(y∣0,K+σ2I)\begin{aligned} \theta^* &= \underset{\theta}{\text{argmax}}\log \mathcal{P}(y|X,\theta) \\ &= \underset{\theta}{\text{argmax}}\log \mathcal{N}(y | 0, K + \sigma^2 I) \end{aligned}θ∗​=θargmax​logP(y∣X,θ)=θargmax​logN(y∣0,K+σ2I)​

from scipy.linalg.lapack import dtrtrs

def log_likelihood(self, X, y):
    n_samples = self.x_train.shape[0]

    K = self.kernel(X)

    Ky = K + self.noise_variance * np.eye(n_samples)

    L = np.linalg.cholesky(Ky)

    LinvY =  dtrtrs(L, y, lower=1)[0]

    logL = - (n_samples / 2.0) * np.log(2 * np.pi)

    logL += - (1.0 / 2.0) * np.square(LinvY).sum()

    logL += -np.log(np.diag(L)).sum()

    return logL

Limitations of GPs

The primary problem is the computational and memory complexity

With a training size NNN and DDD dimensions, here are the complexity estimates for all parts of the GP algorithm:

  • Training: O(N3)\mathcal{O}(N^3)O(N3)
  • Predictive variance: O(N2)\mathcal{O}(N^2)O(N2)
  • Memory: O(ND+N2)\mathcal{O}(ND + N^2)O(ND+N2)

So this gives us a practical limit of about N≈10KN \approx 10KN≈10K training points. There are many solutions to this problem but with the exact GP formulation without any clever tricks, you should be aware of this.
